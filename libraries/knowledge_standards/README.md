## Purpose

This folder contains the governed Knowledge and Standards content pack for the declared VALOR CQV v1.0.1 scope. The operative requirements are internal VALOR/company-worded requirements. External standards are controlled source authorities only and must not be copied into generated outputs.

## Testing-only use

The current K&S package may be used for product testing, dry runs, internal trials, validation of product behavior, and end-to-end workflow testing only when testing metadata is current and the required testing-only stamp is applied.

The current K&S package may not be used for real regulated CQV/GMP output. Regulated output must be blocked, refused, or marked incomplete until user/site source metadata acceptance is complete.

## Source-control rule

External standards are referenced by controlled source IDs, source metadata, and anchors. Their original text is not reproduced. Testing placeholders for source edition, document date, clause number, or locator are not approved regulated-use metadata.
